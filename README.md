This repository contains a curated collection of Kusto Query Language (KQL) queries for Microsoft Defender Advanced Hunting, designed to help security analysts and threat hunters detect, investigate, and respond to potential threats across their environment. It includes queries covering areas such as suspicious process activity, network anomalies, credential access, phishing attempts, persistence techniques, and other security-relevant behaviors. These queries can be used as-is or customized to fit specific organizational needs, enabling faster incident response and improved visibility across endpoints, identities, email, and cloud applications. Users are encouraged to adapt, test, and enhance the queries based on their environment and contribute back to continuously improve the repository.
