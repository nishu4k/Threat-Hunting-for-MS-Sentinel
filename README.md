This repository contains a curated collection of Kusto Query Language (KQL) queries for Microsoft Sentinel, designed to help security analysts and threat hunters detect, investigate, and respond to potential threats across their environment. It includes queries covering areas such as suspicious process activity, anomalous network behavior, credential access attempts, phishing indicators, persistence techniques, and other security-relevant events collected from connected data sources. These queries can be used as-is or customized to align with your organization’s log sources and detection requirements, enabling faster incident response, improved visibility, and proactive threat hunting across your security operations.
