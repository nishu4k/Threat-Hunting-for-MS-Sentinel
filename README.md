Overview

This repository contains a curated collection of Kusto Query Language (KQL) queries designed for Microsoft Defender Advanced Hunting. These queries help security analysts and threat hunters investigate, detect, and respond to potential threats across their environment.

Purpose

The goal of this repository is to provide ready-to-use and customizable KQL queries that can:

Enhance threat detection capabilities
Accelerate incident investigations
Improve visibility across endpoints, identities, email, and cloud apps
Support proactive threat hunting
Contents

You will find a wide range of Defender KQL queries, including but not limited to:

Suspicious process activity
Malicious or unusual network connections
Credential access and privilege escalation attempts
Phishing and email-based threats
Persistence mechanisms
Endpoint anomalies and behavioral detections
Usage
Navigate to the query of interest.
Copy the KQL query.
Open Microsoft Defender Advanced Hunting.
Paste and run the query.
Modify filters (time range, device names, users, etc.) as needed for your environment.
Requirements
Access to Microsoft Defender XDR / Advanced Hunting
Basic understanding of KQL
